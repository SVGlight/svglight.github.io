Guide: Installing Minecraft Fabric Mods

This guide will walk you through setting up your Minecraft Java Edition client to use Fabric mods.

⚠️ YOU NEED A WINDOWS COMPUTER WITH MINECRAFT JAVA FOR THIS.

If you don't own the game yet, you can purchase it here: Purchase Minecraft Java Edition

Step 1: Install Java (JRE)

First, you must install the Java Runtime Environment (JRE) on your computer. Minecraft mods are written in Java, and this software allows your computer to run the `.jar` files used by both the mod installer and the mods themselves.

Download Java (Windows Offline 64-bit)

Step 2: Install the Fabric Mod Loader

The Fabric installer modifies your Minecraft client installation to make it compatible with the Fabric mod ecosystem.

Install the Fabric installer using the link below. Once downloaded, run the `.jar` file and follow the on-screen instructions to create a new Fabric profile in your Minecraft Launcher.

Download Fabric Installer (Universal .JAR)

Step 3: Download Your Mods

You can find a wide variety of Fabric-compatible mods on Modrinth.

Browse Mods on Modrinth

When downloading, make sure you use the sidebar filters on Modrinth to select "Fabric" as the loader and choose the specific Minecraft version you installed in Step 2.

Step 4: Understand Dependencies and Libraries

Many powerful mods rely on separate "library" or "API" mods to function. These required files must also be present in your Minecraft `mods` folder.

For instance, **most Fabric mods require the Fabric API itself** to be in your `mods` folder, alongside the main mods you downloaded. The Fabric API can also be downloaded from the Modrinth link above. Always check the "Dependencies" or "Required Mods" section on a mod's download page for a complete list of necessary library files.

Once all `.jar` files are downloaded, place them into your Minecraft `mods` folder (usually located at `C:\Users\YourName\AppData\Roaming\.minecraft\mods`).
